#webpack-base

установка зависимостей:
```
npm i
```

запуск проекта:
```
npm start
```

сборка проекта в development:
```
npm run dev
```

сборка проекта в production:
```
npm run build
```

особенности сброки:
```
Babel
Html / Pug
Sass / Scss / Less
Autoprefixer
CSSNano
Normalize.css
```

псевдонимы:
```
alias for images folder: ~img
alias for fonts folder: ~fonts
```
